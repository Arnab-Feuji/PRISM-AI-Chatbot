Agent Routing Logic
LangGraph orchestration flow for every patient conversation.
PRIMARY AGENT FLOW
1. Detect language (EN/HI/TE/ES/PA)
2. Translate → English
3. Retrieve from agent's exclusive DB
4. Primary agent responds
5. Score confidence + frustration
6. Format + translate response
SPECIALIST ESCALATION
Triggered when:
Confidence score < 0.70

Agent: {CA1→CA1-S} {DM1→DM1-S}
{CV1→CV1-S} {MH1→MH1-S} {RS1→RS1-S}

Temperature: 0.10 (more precise)
HUMAN ESCALATION
Triggered when:
Frustration score > 75

Agent: {CA1→CA1-H} {DM1→DM1-H}
{CV1→CV1-H} {MH1→MH1-H} {RS1→RS1-H}

Temperature: 0.50 (more empathetic)
HARD GUARDRAIL OVERRIDES
CV2: Any chest pain → 911 first
MH5: Suicidal ideation → crisis line
DM1: DKA signs → ER immediately
RS1: Severe attack → 911 + CPR
CA2: Febrile neutropenia → ER
